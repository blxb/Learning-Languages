# Learning-Languages
This script is made for learning other languages.. You will be able to setup any language you really want.

# How to setup other languages??
1. Go to the langEducation.py
2. change the words_by_level dictionary to your desired language (DO NOT TOUCH DIFFICULITY LEVELS(unless you know what you are doing), you can only add or remove them)
3. in line 86 tell in print that now it is translating to your desired language that you want user to educate with
4. in line 91 translator.translate(text, dest="ru") change the dest to your language using language code.. In this case "ru" stands for Russian...
5. You can discover your language code on internet or use availableLanguages list that contains some of those language codes.

# And now enjoy your first proper language education program..
